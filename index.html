HTML Introduction....

HTML stands for Hypertext Markup Language used to create web pages.HTML will structure content of the web page. Markup language means that consists of set of tags. Hypertext means having links to other text elements.HTML code is executed by web browsers such as Google Chrome, Mozilla Firefox, Safari, etc.
HTML documents are plain text files with a .html extension. HTML was created by Tim Berners-Lee in 1991.Trained by Mr. Ratan Sir. HTML maintained by W3C
